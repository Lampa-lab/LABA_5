# Гальцов Денис РИ-241101

Данный репозиторий предлагет решение на 4 вариант работы в Unity

Решение построено по принципу многослойности, где каждый кружок находится в своей плоскости и взаимодействует только с объектами в своей или дополнительной плоскости.
Для создания различных взаимодействий объектов с объектами были добавлены следующие компоненты:
1)Физика взаимодействия объектов RigidBody 2D, а также настроена колизия объектов
2)Материалы, для различного взаимодействия объектов (отскок)
3)Различные слои объектов, чтобы избежать столкновения некоторых объектов по условию задачи
